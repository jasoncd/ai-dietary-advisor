# Push Your AI Dietary Advisor to GitHub

## Step 1: Create GitHub Repository

1. Go to [github.com](https://github.com)
2. Click the "+" icon → "New repository"
3. Repository name: `ai-dietary-advisor` (or any name you prefer)
4. Description: `AI-powered dietary advisor with social media commenting system`
5. Choose "Public" (required for free deployments)
6. **Don't** check "Initialize with README" (we already have files)
7. Click "Create repository"

## Step 2: Connect Your Project to GitHub

Copy the commands GitHub shows you, they'll look like this:

```bash
git remote add origin https://github.com/YOUR_USERNAME/ai-dietary-advisor.git
git branch -M main
git push -u origin main
```

**Replace `YOUR_USERNAME` with your actual GitHub username**

## Step 3: Push Your Code

After running the commands above, your code will be on GitHub!

## Step 4: Deploy to Free Hosting

Once on GitHub, you can deploy to:

### Railway (Recommended - Best Free Option)
1. Go to [railway.app](https://railway.app)
2. "New Project" → "Deploy from GitHub repo"
3. Select your `ai-dietary-advisor` repository
4. Add PostgreSQL database
5. Your app goes live automatically!

### Alternative: Render
1. Go to [render.com](https://render.com)
2. "New Web Service" → Connect GitHub
3. Select your repository
4. Build: `npm run build`
5. Start: `npm start`
